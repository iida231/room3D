# Room Planner 3D

部屋の大きさを自由に設定し、家具の配置を3Dで確認できるWebアプリです。

## 動作環境

- Node.js 18 以上
- モダンブラウザ (Chrome / Firefox / Edge 最新版)

## セットアップ

```bash
# 依存パッケージのインストール
npm install
```

## 起動

```bash
npm run dev
```

起動後、ブラウザで http://localhost:5173 を開いてください。

## 主な機能

### 部屋の設定
- 幅・奥行き・高さを cm 単位で設定
- 床と壁の色をカラーピッカーで変更

### 建築要素
- **窓** — 任意の壁（北/南/東/西）の任意の位置に追加。幅・高さを設定可能
- **ドア** — 任意の壁の任意の位置に追加。幅・高さを設定可能

### 家具
| 種類 | デフォルトサイズ (W × D × H cm) |
|------|--------------------------------|
| エアコン | 80 × 25 × 30 |
| 机 | 120 × 60 × 75 |
| クローゼット | 90 × 60 × 200 |
| 棚 | 90 × 30 × 120 |
| 本棚 | 80 × 30 × 180 |
| 直方体（カスタム） | 50 × 50 × 50 |

### 操作方法
- **回転** — 左ドラッグ
- **ズーム** — マウスホイール
- **移動** — 右ドラッグ
- **選択** — オブジェクトをクリック（または左の一覧をクリック）
- **プロパティ編集** — 選択後、右パネルで位置・サイズ・回転・色を変更

### その他
- 天井の表示/非表示を切り替え可能
- 配置データはブラウザの LocalStorage に自動保存され、再読み込み後も復元されます
- 「リセット」ボタンで初期状態に戻せます

## ビルド（本番用）

```bash
npm run build
```

`dist/` フォルダに静的ファイルが生成されます。任意のWebサーバーに配置して使用できます。
